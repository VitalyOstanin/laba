laba

Desktop tray client and command-line interface for OpenProject — built with Tauri and a shared Rust core.

View on GitHub Download releases

What it does

Native desktop tray app

Work packages, notifications and logged time for OpenProject in a small tray application for Linux, Windows and macOS.

Scriptable CLI

A laba command-line binary with JSON output by default, for automation and scripting against the OpenProject API.

Multiple backends

Several OpenProject servers with a selectable default, and GitHub issues and notifications aggregated in the same dashboard.

Open source

MIT-licensed, a single Rust workspace. The desktop app talks to the OpenProject API directly through the core library.

Get started

Command-line interface:

laba server add my-op https://op.example
laba auth login --server my-op
laba wp list

The desktop app is distributed as release builds (including a Debian package) on the releases page. Build instructions are in the README.
