laba

Desktop tray dashboard and command-line interface for GitHub — issues, pull requests and notifications in one place. Built with Tauri and a shared Rust core.

View on GitHub Download releases

What it does

GitHub in your tray

Your GitHub issues, pull requests and notifications in a small tray application for Linux, Windows and macOS. Uses the gh CLI, so no extra token setup.

Scriptable CLI

A laba command-line binary with JSON output by default, for automation and scripting — GitHub is the default backend.

Pluggable backends

One dashboard and CLI over a multi-backend design. Besides GitHub, OpenProject is implemented too; more trackers (Jira, YouTrack, GitLab, …) fit the same facade.

Open source

MIT-licensed, a single Rust workspace. The desktop app talks to each backend directly through the core library.

Get started

Command-line interface (GitHub is the default backend):

laba server add gh github.com
laba wp list

The desktop app is distributed as release builds (including a Debian package) on the releases page. Build instructions are in the README.
